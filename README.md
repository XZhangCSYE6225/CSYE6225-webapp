## Initialize Steps
1. Set up a workspace
```sh
cd ~/
mkdir -p ./workspace/csye6225
```
2. Clone the repo
```sh
git clone git@github.com:XZhangCSYE6225/webapp.git
```
3. Get into the directory of the repo
```sh
cd ./webapp
```
4. Install all dependencies
```sh
npm i
```
5. Run the app
```sh

1
nodemon server.js
```
